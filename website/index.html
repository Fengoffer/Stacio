CAPABILITIES

终端管理、分屏协作、同步执行、文件编辑、SCP 传输、内置浏览器、设备告警和 AI Agent——所有远程操作能力收进一个原生 macOS 工作台。

real-time edit · dual backup · syntax highlight

WORKFLOW

从连接到解决，全程在一个工作台。

Stacio 把 SSH 连接、分屏终端、文件传输、远程编辑、隧道、设备监控和 AI 诊断放进同一个操作闭环，告别多工具来回切换。

01

Connect

从保存会话快速连接，SSH/Telnet/串口三合一，会话分组一目了然。

02

Inspect

终端分屏查看多台设备上下文，远程目录、日志和状态同屏展示。

03

Transfer

通过内置 SCP 工作流上传下载文件和文件夹，传输队列实时可查。

04

Tunnel

创建并监控 SSH 隧道，local / remote / dynamic forward 集中管控。

05

Resolve

AI Agent 结合终端上下文生成排查建议，本地 Agent 协助分析和修复。

SECURITY

本地优先，敏感操作始终由用户确认。

Stacio 把远程工作流收进同一个 macOS 应用，但凭据、会话和敏感动作仍遵循本地优先与显式确认。

凭据交给 macOS Keychain，会话数据和本地 Agent 调用记录均保存在本地，不默认外传。
AI 建议和本地 Agent 生成的命令均需用户确认后执行，敏感操作始终受控。
诊断、日志和设备信息默认脱敏，减少复制粘贴排查时的泄露风险。

UPDATE NOTES

Stacio 0.13.3

本次正式版优化本地 Agent 与终端协作体验，并提升远程会话稳定性。

主要更新

  • 本地 Agent 能识别终端审计完成标记；即使标记分散在多段输出中，也会在命令结束后自动继续并返回总结。
  • 改善终端输出回传和会话完成状态，减少排查任务停留在等待或需要手动确认的情况。
  • 提供独立的 Apple Silicon 与 Intel Mac 安装包，并在发布前校验 App、CLI、VNC、Core 与 Sparkle 组件架构。
  • 优化远程会话、文件面板与 AI 助手的稳定性。

下载

  • Apple Silicon：Stacio-0.13.3-arm64.dmg
  • Intel Mac：Stacio-0.13.3-x86_64.dmg

安装提示

  • 支持 macOS 14 及以上版本。
  • 当前版本为未公证的 ad-hoc 签名包；首次打开如被 macOS 拦截，请在 Finder 中右键 Stacio.app 并选择“打开”。
  • 已保存的会话、凭据引用、AI 配置、传输记录和应用设置会继续保留。
